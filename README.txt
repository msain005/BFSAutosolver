** To compile this program there is a makefile in which you can compile on terminal with "make" then "java Puzzle"

** This program is an autosolve for any valid input for a 15 puzzle including 0. If it does not have
   a zero then a solution will not be found. The input MUST have a 16 character input, if any lower
   it will prompt for more values till 16, if more it will ignore inputs after the 16th input.

** When ran, this program traces each Phase in the puzzle in which it moves towards the Goal.
